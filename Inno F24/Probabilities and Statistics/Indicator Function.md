## Definition
- Indicator [[Function]] is a [[Random Variable]] that returns $1$ for some given [[Event (Probability)|Event]].
$$I_A(\omega) = \begin{cases}
1, \omega \in A\\
0, \text{otherwise}
\end{cases}$$
## Notation
- For event $A$: $$I(A)(\omega)\text{ or } I_A(\omega)$$
